# Dapp Chain

An ethereum product for all the logistics and management, a generic supply chain.



## Installation

Use the package manager [yarn](https://yarnpkg.com/).

```bash
yarn install
yarn start
```
run this in different terminal...
```
yarn chain
```
run this in different terminal...
```
yarn deploy
```
